# MVP Generator

## Overview
MVP Generator is a tool designed to streamline the process of creating a Minimum Viable Product (MVP). It helps project managers and leads to define the project flow, identify the easiest path to completion, and manage team resources effectively.

## Features

- **MVP Flow Generation**: Automatically generates a structured flow for your project's MVP.
- **Optimization**: Identifies the easiest and most efficient way to achieve the project goals.
- **Resource Management**:
    - Prompts for the number of available developers.
    - Assigns specific tasks to each developer based on the project requirements.
- **Timeline Management**:
    - Generates a weekly schedule to ensure the project is completed within the targeted timeframe.

## How It Works

1. **Define Project**: Input your project details and goals.
2. **Generate Strategy**: The tool calculates the optimal flow.
3. **Allocate Team**: Enter the number of developers in your team.
4. **Get Plan**: Receive a detailed task assignment and a weekly roadmap for your team.
